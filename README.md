This repository contains the source and content for [Fedora Status](https://wwww.fedorastatus.org). It uses the Pelican static site generator.

Read the [Fedora Infrastructure Standard Operating Procedure (SOP)](https://docs.fedoraproject.org/en-US/infra/sysadmin_sops/status-fedora) for details on how to add, move, and edit outage notices, as well as publishing changes live.
